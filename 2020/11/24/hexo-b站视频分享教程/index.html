hexo插件安装插入视频代码块及音频

1.hexo插件安装

使用 Hexo 插件插入音乐/视频
两个好用的hexo插件：

**hexo-tag-aplayer：https://github.com/grzhan/hexo-tag-aplayer)

**hexo-tag-dplayer： https://github.com/NextMoe/hexo-tag-dplayer

1.1 播放音乐的aplayer

在cmd页面内，使用npm安装：

npm install hexo-tag-aplayer

1.2 播放视频的dplayer

在cmd页面内，使用npm安装：
npm install hexo-tag-dplayer
